# Professional Academic Portfolio - Jekyll Site

A clean, professional Jekyll site for academics and professionals.

## Features

- **Academic Aesthetic**: Navy blue and gold color scheme with elegant typography
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **SEO Optimized**: Includes jekyll-seo-tag plugin for search engine optimization
- **Modular Structure**: Easy to customize and extend
- **Navigation**: Sticky header with links to Home, About, Research, Publications, and CV
- **Blog-Ready**: Ready for publishing research updates and articles

## Quick Start

1. Clone or download this repository
2. Install dependencies: `bundle install`
3. Run the development server: `bundle exec jekyll serve`
4. Open `http://localhost:4000` in your browser

## Customization Guide

See README.md in the root for detailed customization instructions.

## File Structure

```
├── _config.yml          # Site configuration
├── _includes/           # Reusable HTML components
├── _layouts/            # Page layout templates
├── _posts/              # Blog posts
├── assets/
│   ├── css/            # Stylesheets
│   └── images/         # Image assets
├── index.md            # Home page
├── about.md            # About page
├── research.md         # Research page
└── publications.md     # Publications page
```
